
# E-Commerce Funnel Analysis

## Overview
Analyzed an e-commerce conversion funnel using Python to identify where users drop off and what drives purchases.

## Key Findings
- Overall conversion rate: 3.2% (industry average: 2–4%)
- 81.8% of users dropped off before adding to cart
- 100% of purchases came from referred traffic — direct visitors bought nothing
- Multi-session users convert at 2.4x the rate of single-session users

## Tools Used
- Python, Pandas, Matplotlib
